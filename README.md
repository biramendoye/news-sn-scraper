# News SN Scraper

### Description

A web scraper that allows us to scrape data from information sites located in Senegal.
Below are the news sites used:
- [senego.com](http://senego.com)
- [seneweb.com](http://seneweb.com)

### Tech / Framework

Built with :
- BeautifulSoup
- requests
